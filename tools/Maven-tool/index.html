# Maven 安装

要是新版本 IDEA 则不需要修改 Maven , IDEA 中默认整合了 Maven ，只不过是版本问题

# 安装

下载地址： http://maven.apache.org/

目录结构：

bin：存放的是执行文件，命令
在 IDEA 中可以直接集成 Maven:

conf 目录：下面有一个非常重要的配置文件 --》settings.xml---》maven 的核心配置文件 / 全局配置文件。

** 注：** 如果没有 .m2 目录 ，自己手动执行 mvn 命令：

mvn help:system
