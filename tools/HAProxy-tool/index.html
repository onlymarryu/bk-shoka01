# 1、准备好 HAProxy 的安装包

https://src.fedoraproject.org/repo/pkgs/haproxy/

# 2、解压到 /usr/local 目录
# 3、进入到解压后的目录，查看内核版本，进行编译
# 4、编译完成之后，开始进行安装
# 5、安装完成之后，创建目录，创建 HAProxy 配置文件

6、向配置文件中添加配置信息

  • stats uri /admin 账号
  • stats auth admin:123123 密码
# 7、启动 haproxy 服务
# 8、查看 haproxy 的进程，如果存在则说明没有问题
# 9、打开浏览器访问，用户名为 admin，密码为 123123
